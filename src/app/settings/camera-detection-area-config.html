Detection Zone Setup

Currently this interface is very crude. It will be improved later. Click on the upper left of the zone you would like detection to occur in, then the lower right. The coordinates will be used to define a rectangle which will be used for detection. Currently no status is shown during clicking, but after two coordinates are clicked, you will see the coordinates used and be given the option to accept.
